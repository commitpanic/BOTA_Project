# Lato Fonts

This directory contains Lato font files used for PDF generation in the BOTA project.

## Font Files

- **Lato-Regular.ttf** - Regular weight
- **Lato-Bold.ttf** - Bold weight
- **Lato-Italic.ttf** - Italic style
- **Lato-BoldItalic.ttf** - Bold Italic style

## Source

Fonts are downloaded from the official Google Fonts repository:
https://github.com/google/fonts/tree/main/ofl/lato

## License

Lato is licensed under the SIL Open Font License (OFL):
https://scripts.sil.org/OFL

## Purpose

These fonts are used to ensure proper rendering of Polish characters (ą, ć, ę, ł, ń, ó, ś, ź, ż) 
in PDF diploma certificates generated by ReportLab.

## Installation

To download/update fonts, run:
```bash
python download_lato_fonts.py
```

## Usage

The fonts are automatically registered in `frontend/views.py` when generating diploma PDFs.
If the fonts are not found, the system falls back to Helvetica.
